## 🎯 功能亮点

### 1. 智能相关度算法
```
得分 = 基础分 + 文本类型分 + 匹配次数分 + 长度因子分

- 标题类型: +50分
- 描述类型: +30分
- 表格类型: +20分
- 完全匹配: +100分
- 包含关键词: 每次+10分
```

### 2. 同义词系统
支持动态扩展的同义词词典：
```typescript
{
  '油门踏板': ['踏板位置传感器', 'APS', ...],
  'ECU': ['电控单元', 'Electronic Control Unit', ...]
}
```

### 3. 性能优化
- PDF文本提取结果缓存
- 按需加载PDF页面
- 正则表达式优化搜索
- Nginx静态资源缓存
